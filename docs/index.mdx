# Build powerful AI agents with IBM i database integration

The **IBM i MCP Server** is a production-grade TypeScript implementation of the Model Context Protocol (MCP) specifically designed for IBM i systems. It provides AI agents with secure, type-safe access to IBM i databases through advanced SQL tools, dynamic query execution, and comprehensive observability features.

## Architecture Highlights

<Note>
The server implements a unique **"Logic Throws, Handler Catches"** architectural pattern that ensures clean separation of concerns, comprehensive error handling, and excellent testability.
</Note>

### Core Components

<Tabs>
<Tab title="Transport Layer">
- **Stdio Transport**: Direct process communication for development and testing
- **HTTP Transport**: Production-ready streamable server with session management
- **Authentication**: Configurable JWT, OAuth, or no-auth modes
</Tab>

<Tab title="SQL Tools System">
- **Dynamic Tools**: Runtime SQL execution with security restrictions
- **YAML Tools**: Predefined SQL statements with parameter templates
- **Toolsets**: Organized collections of related tools for different use cases
</Tab>

<Tab title="IBM i Integration">
- **Connection Pooling**: Efficient database connection management
- **Mapepire Integration**: Modern IBM i database connectivity
- **Security**: Built-in SQL restrictions and input validation
</Tab>
</Tabs>

## Getting Started

<Steps>
<Step title="Install Dependencies">
  Clone the repository and install the required Node.js packages.
  
  ```bash
  git clone https://github.com/ajshedivy/ibmi-mcp-server.git
  cd ibmi-mcp-server
  npm install
  ```
</Step>

<Step title="Configure IBM i Connection">
  Set up your IBM i database connection details in the `.env` file.
  
  ```bash
  cp .env.example .env
  # Edit .env with your IBM i connection details
  ```
</Step>

<Step title="Build and Run">
  Build the project and start the server in your preferred transport mode.
  
  ```bash
  npm run build
  npm run start:http  # or start:stdio
  ```
</Step>
</Steps>

<Check>
Your IBM i MCP Server is now ready to provide AI agents with secure database access!
</Check>
